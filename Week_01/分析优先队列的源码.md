# 概要
1. jdk版本： 1.8
2. 路径： java.util.PriorityQueue
3. 继承： java.util.AbstractQueue
4. 实现： java.io.Serializable
5. 默认初始容量： 11

# 数据结构
1. 队列
2. 用数组实现

# 方法
1. grow：当超过容量时的扩容
    1. 原容量小于64，在原容量上加2
    2. 原容量大于等于64，翻倍
2. 入队
    1. add 调用offer
    2. offer
        1. 添加null 会造成npe
        2. 返回是否成功入队
        3. 使用队列的比较器或元素的比较器
            1. siftUpUsingComparator 队列比较器
            2. siftUpComparable 元素的比较器
        4. 需要进行二分查找来排序，时间复杂度O(logN)
    3. peek 查看队首元素 时间复杂度O(1)
    4. poll
